If else é uma estrutura de controle que permite com base na condição declarada direcionarmos a aplicação/sistema/site/app


if else simples - Sintaxe

if(declaração verdadeira) se a condição for verdadeira

else(declaração falsa) se a condição for falsa


if else composta - Sintaxe

if(declaração verdadeira) se a primeira condição for verdadeira

if else(declaração verdadeira) se a segunda condição for verdadeira

else(declaração falsa) se a condição for falsa


Operadores de comparação

Igual (==) - verifica se os valores são iguais

Idêntico (===) - verifica se os valores são iguais e do mesmo tipo

Diferente (!=) - verifica se os valores são diferentes

Igula (!==) - verifica se os valores são iguais e de tipos diferentes

Menor (<) - verifica se os valor da esquerda é menor que o valor da direita

Maior (>) - verifica se os valor da esquerda é maior que o valor da direita

Menor igual (<=) Verifica se o calor da esquerda é menor ou igual ao valor da direita

Maior igual (>=) Verifica se o calor da esquerda é maior ou igual ao valor da direita
